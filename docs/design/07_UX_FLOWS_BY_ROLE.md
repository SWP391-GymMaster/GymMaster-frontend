# 07 — UX Flows by Role

## 1. Admin flow

Admin chịu trách nhiệm quản trị toàn hệ thống.

```text
Login
→ Admin Dashboard
→ Manage Users/Staff/PT/Member
→ Manage Packages
→ Assign PT
→ View Dashboard
→ View Audit Logs
```

Admin dashboard cần:

- revenue
- active members
- expired members
- today check-ins
- pending payments
- recent audit logs

## 2. Staff flow

Staff là người thao tác nhanh tại quầy.

```text
Login
→ Staff Dashboard
→ Search Member
→ View Membership Status
→ Sell/Renew Package
→ Record Payment
→ Check-in Member
```

UX ưu tiên:

- search nhanh
- ít bước
- status rõ
- action lớn dễ click
- check-in feedback tức thì

## 3. PT flow

PT quản lý member được phân công.

```text
Login
→ PT Dashboard
→ Assigned Members
→ Member 360 Profile
→ Create Workout Plan
→ Add Trainer Note
→ View Progress
→ View Nutrition History
```

PT không thấy revenue/payment.

## 4. Member flow

Member dùng app đơn giản.

```text
Login
→ Member Dashboard
→ View Membership
→ Show QR / Check-in status
→ View Workout Plan
→ View Trainer Notes
→ Add Progress
→ Add Meal Log
→ View Calorie Summary
```

Member menu nên ít và dễ hiểu.

## 5. Demo flow chính

```text
Admin/Staff login
→ Create/find Member
→ Sell Package
→ Member check-in
→ Admin assign PT
→ PT add plan/note
→ Member add meal log
→ Admin view dashboard/audit
```
